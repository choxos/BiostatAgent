# itc-modeling

Expert agents for indirect treatment comparison modeling in R, covering pairwise
meta-analysis, network meta-analysis, MAIC, STC, and ML-NMR — with tidy modeling
practices and NICE DSU alignment.

**7 agents · 2 commands · 6 skills**

## Agents

- `itc-architect` — entry point; routes to the appropriate ITC method.
- `pairwise-meta-analyst` — fixed/random effects, heterogeneity, publication bias.
- `nma-specialist` — frequentist (`netmeta`) and Bayesian (`gemtc`) NMA, league tables.
- `maic-specialist` — Matching-Adjusted Indirect Comparison via `maicplus`.
- `stc-specialist` — Simulated Treatment Comparison (anchored / unanchored).
- `ml-nmr-specialist` — Multilevel Network Meta-Regression via `multinma`.
- `itc-code-reviewer` — methodology + R code review for ITC analyses.

## Commands

- `/itc-analysis` — guided ITC analysis workflow.
- `/itc-review` — review existing ITC code and outputs.

## Skills

`tidy-itc-workflow`, `pairwise-ma-methodology`, `nma-methodology`,
`maic-methodology`, `stc-methodology`, `ml-nmr-methodology`.
