# bayesian-modeling

Create, review, and validate Bayesian models in Stan, PyMC, JAGS, and WinBUGS.

**6 agents · 3 commands · 9 skills**

## Agents

- `model-architect` — entry point that routes Bayesian modeling tasks to the right specialist.
- `stan-specialist` — Stan 2.37 syntax, HMC/NUTS, all program blocks, `cmdstanr`.
- `pymc-specialist` — PyMC 5 syntax, samplers, ArviZ diagnostics integration.
- `bugs-specialist` — WinBUGS / JAGS, precision parameterization, R integration.
- `model-reviewer` — reviews specs for correctness, efficiency, prior choice.
- `test-runner` — runs short chains on synthetic data and reports convergence.

## Commands

- `/create-model` — interactive workflow for new models.
- `/review-model` — review and improve an existing model.
- `/run-diagnostics` — execute with test data and report MCMC diagnostics.

## Skills

`stan-fundamentals`, `pymc-fundamentals`, `bugs-fundamentals`,
