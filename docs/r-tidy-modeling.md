# r-tidy-modeling

Expert agents for R data science and biostatistics covering data wrangling,
feature engineering, model building, statistical evaluation, clinical trials,
genomics, documentation, and reproducible reporting using `tidyverse` and
`tidymodels`.

**10 agents · 7 commands · 23 skills**

## Agents

`r-data-architect`, `tidymodels-engineer`, `feature-engineer`,
`biostatistician`, `data-wrangler`, `viz-specialist`, `reporting-engineer`,
`r-code-reviewer`, `r-docs-architect`, `r-tutorial-engineer`.

`/r-analysis`, `/r-code-review`, `/r-model-comparison`,
`/r-clinical-analysis`, `/r-project-setup`, `/r-doc-generate`,
